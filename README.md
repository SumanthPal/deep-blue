# Deep Blue

## Overview

A flask project w/ bluetooth integration
## Setup

1. Create a virtual environment and activate it:
    ```sh
    python3 -m venv venv
    source venv/bin/activate
    ```

2. Install the dependencies:
    ```sh
    pip install -r requirements.txt
    ```

3. Run the application:
    ```sh
    flask run
    ```
